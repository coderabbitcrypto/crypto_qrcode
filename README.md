# crypto-qrcode

> Simple SPA(Single Page Application) the will generate for us an QR Code image to be read by our phones/tablets to get the address and amount to be paid.

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
